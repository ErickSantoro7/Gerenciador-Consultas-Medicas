# MongoDB-CRUD-Aggregation
Coleção de comandos MongoDB (shell) para gerenciamento de dados de consultas médicas. O projeto cobre:  C.R.U.D: Inserção (insertMany), busca (find) e atualização (updateMany).  Modelagem NoSQL: Exemplos de documentos Integrados e Normalizados.  Agregação: Pipelines complexos com $group (contagem) e $lookup (simulando JOINs) para análise de dados.
